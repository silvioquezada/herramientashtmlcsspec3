Enlaces de Referencias de Contenido

Las imágenes, fotos, íconos, texto se extrajeron de diferentes páginas web, por lo que este sitio web está diseñado con fines educativos, por lo que cada recurso extraído se referencia de los siguientes enlaces:

Imágenes de slider fue extraído:

https://www.pexels.com/es-es/

Las imágenes del slider en el portal fue extraído del sitio web pexels, ya que cuenta con variedades de imágenes en su mayoría son gratuitas, y se seleccionó solo las imágenes de tipo gratuitas.

Íconos del sitio web fue extraído:

https://iconos8.es/

Las íconos del sitio web fueron extraídos del sitio web icons8, este sitio web icons8 ofrece una variedad de íconos gratuitos solo hasta 100px, mayor a 100px tiene un costo por lo que habría que pagar para utilizarlos, solo se utilizó íconos hasta 100px máximos, íconos para las secciones de las páginas ubicadas en el portal como sección de categorías, sección de presentación y sección de enlaces, íconos para los lugares por categorías y detalles de cada lugar como la ubicación y el calendario.

Imágenes y contenido de las categorías fueron extraídos del sitio:

https://www.goraymi.com/es-ec/el-oro/guia-turismo-d2cf17

Todo el contenido de cada lugar turístico ya sea en categorías y detalle de imágenes y contenido fue extraido del sitio web goraymi, este sitio web ofrece mucha información de muchos lugares turísticos, todo el contenido publicado de goraymi lo hacen personas que quieren hacer algún tipo de publicidad, por lo que se utilizó para extraer el contenido e imágenes reales de los diferentes lugares, se hace referencia a este sitio web para respetar los derechos de autor del contenido obtenido.
